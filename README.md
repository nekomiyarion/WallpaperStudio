## 使い方 (How to use)

1. [最新のリリースページ](https://github.com/nekomiyarion/WallpaperStudio/releases/latest)から `WallpaperStudio_v4.0_Windows.zip`（ご自身で命名したZIP名）をダウンロードしてください。
2. ZIPファイルをすべて展開（解凍）します。
3. `WallpaperStudio.exe` を実行します。
   - 初回起動時に、実行ファイルと同じ場所に **`movies`** フォルダと **`background`** フォルダが自動生成されます。

## 動画の追加方法 (Adding Videos)

お好みの動画を再生したい場合は、以下の手順でファイルを作成したフォルダに入れてください。

* **動画ファイル**: `movies` フォルダに入れます。
* **ジャケット画像**: `background` フォルダに入れます。

> [!IMPORTANT]
> **再生のポイント**
> 動画ファイルとジャケット画像（プレビュー用）の**ファイル名を同じにする**必要があります。
> 例：`sample_video.mp4` を使う場合、画像は `sample_video.jpg`（またはpng）という名前にしてください。
